public void verifyAccount (String verificationCode, String signalingKey, boolean supportsSms, int axolotlRegistrationId)

Verify a TextSecure account.

Parameters
verificationCode The verification code received via SMS or Voice (see requestSmsVerificationCode() and requestVoiceVerificationCode()).
signalingKey 52 random bytes. A 32 byte AES key and a 20 byte Hmac256 key, concatenated.
supportsSms Indicate whether this client is capable of supporting encrypted SMS.
axolotlRegistrationId A random 14-bit number that identifies this TextSecure install. This value should remain consistent across registrations for the same install, but probabilistically differ across registrations for separate installs.
