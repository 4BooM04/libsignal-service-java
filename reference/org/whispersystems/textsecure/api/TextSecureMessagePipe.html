public TextSecureEnvelope read (long timeout, TimeUnit unit, TextSecureMessagePipe.MessagePipeCallback callback)

A blocking call that reads a message off the pipe (see read(long, java.util.concurrent.TimeUnit) Unlike read(long, java.util.concurrent.TimeUnit), this method allows you to specify a callback that will be called before the received message is acknowledged. This allows you to write the received message to durable storage before acknowledging receipt of it to the server.

Parameters
timeout The timeout to wait for.
unit The timeout time unit.
callback A callback that will be called before the message receipt is acknowledged to the server.

  • The message read (same as the message sent through the callback).
